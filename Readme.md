## 🧠 Project Overview

🔄 **Loop Trap** is not just a game… it’s a psychological experience.

You are given choices —
but every choice leads you back to the same place.

👉 Or does it…?

---

## 🎮 Experience

👁️ You see two doors
🚪 You choose one
🔁 You return again

The loop continues…
until you realize something is **hidden**.
